Gets a listing of Picasa/Google+/Flickr Web albums and displays album thumbnails on your page.

== Description ==

SlideshowFx will take a given Picasa username or Flickr user ID and fetch a listing of all public albums for the given user. Thumbnails to the individual albums will be displayed in a table on the specified WordPress page.

Please note that this plugin requires PHP5 or higher and cURL library.

== Installation ==
= Automatic install =
1. From WordPress back-end select menu Plugins > Add New
2. Select Upload, choose SlideshowFx zip file and click on Install Now
3. Click on Active Plugin
4. Configure your Picasa username or Flickr user ID in SlideshowFx parameters (you can keep default value for a test)
5. Place [sfx] in the content of the page

= Manual install =
1. Extract zip file
2. Upload the 'slideshoxfx' directory to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure your Picasa username or Flickr user ID in SlideshowFx parameters (you can keep default value for a test)
5. Place [sfx] in the content of the page
